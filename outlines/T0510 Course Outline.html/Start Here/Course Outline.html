Course Description

This course is intended for programmers with experience in the .Net framework with C# and basic front-end skills who are interested in developing and further enhancing the skills necessary to build dynamic web applications. The course focuses on creating applications with ASP.NET Core to build full stack Single Page Applications and REST APIs (Which is an architectural pattern to create web services)

Course Rationale

At the end of the course, students will be able to build a complete Single Page Application that involves both front end, back end and a database and, therefore, experience what a real application looks like in practice. This course combines the use of all the knowledge gained in previous courses.

Course Outcomes

Students who successfully complete this course will be able to:

  1. Write well-structured, easily maintained, dynamic single page web applications with ASP.NET Core.

  2. Create discoverable, self-documenting REST APIs with ASP.NET Core and Swagger. 

  3. Demonstrate how to persist web application data using Entity Framework.
  4. Demonstrate how to debug ASP.NET Core web applications, making use of good practice and debugging tools.
