Course Description

Embark on a transformative journey in our Front-End Frameworks Course, where you will study the intricacies of Angular and React. First, you will examine Angular's robust architecture, mastering TypeScript, standalone components, and reactive forms. Then, you will study React's declarative paradigm, functional components with hooks, and virtual Document Object Model for efficient user interface creation. Using Microsoft Azure, you will develop your skills in cloud-based deployment and, through hands-on projects, delve into state management, routing, client-side validation, and industry best practices. This course will give you a comprehensive understanding of these frameworks, enabling you to craft an immersive user experience and navigate the ever-evolving landscape of web development.

Course Rationale

Students will learn to develop dynamic Single Page Web Applications using today’s most popular front-end frameworks: Angular and React. Through a series of graduated assignments, students will develop an application called The Everyday Market App in each of the two frameworks. The Everyday Market App contains all the components of a standard shopping cart app and will serve as a valuable portfolio piece for students upon graduation.

Course Outcomes

Students who successfully complete this course will be able to:

  1. Demonstrate how to build a single page application using Angular and .NET.

  2. Demonstrate how to build a single page application using React and .NET.

  3. Demonstrate how to implement Routing and form validation in Angular and React.
  4. Demonstrate how to deploy web applications to the MS Azure cloud using modern CI/CD practices.
