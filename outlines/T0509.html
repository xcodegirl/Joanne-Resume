Course Description

This course builds on the C# programming course, providing you with lessons on how to develop cross-platform applications using .NET. We explore new features such as working with Files, Streams and Serialization, and working with data using Entity Framework core. Additionally, we explore how data is secured using encryption and hashing.  Finally, we will learn how to use LINQ syntax for querying data collections.

Course Rationale

In today's competitive technological landscape, the ability to develop scalable, efficient, and modern applications is a valuable skill set. This course has been constructed to provide participants with understanding of the .NET framework, empowering them to build diverse applications ranging from desktop and web applications to cloud-based services.

Course Outcomes

Students who successfully complete this course will be able to:

  1. Utilize fundamental .NET data types: numbers, text, collections, arrays to construct a function.

  2. Demonstrate how to Publish .NET apps and libraries for use cross-platform.

  3. Demonstrate how to Read and write to files and streams, text encoding and serialization.
  4. Read and write to databases, such as Microsoft SQL Server, SQLite, and Azure Cosmos DB, using the object-relational mapping technology named Entity Framework Core (EF Core).
  5. Utilize Language Integrated Query (LINQ), a set of language extensions that add the ability to work with sequences of items and then filter, sort, and project them into different outputs.
