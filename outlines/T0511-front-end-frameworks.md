**Program:** Full Stack Development (FSD)

**Delivery:** Online Instructor-Guided  
**Educational Developer:** Yi Jin · **SME:** Karan Aggarwal

## Description

Embark on a transformative journey studying Angular and React. Examine Angular's robust architecture, mastering TypeScript and component-based development. Study React's declarative paradigm and virtual DOM for efficient UI creation. Using Microsoft Azure, develop skills in cloud-based deployment; through hands-on projects, delve into state management, routing, and industry best practices.

**Rationale:** Students develop dynamic Single Page Web Applications using today's most popular front-end frameworks. Through a series of graduated assignments, students build *The Everyday Market App* — a standard shopping-cart application — in both Angular and React, serving as a portfolio piece upon graduation.

---

## Learning Outcomes

**Outcome 1: Build a single-page application using Angular and .NET**
- Describe the Angular framework
- Create Angular applications using the Angular CLI
- Construct reusable modules and components
- Bind data into web elements using ngModel
- Manage state and component interaction
- Implement routing with the Angular Router
- Validate forms using client-side validation

**Outcome 2: Build a single-page application using React and .NET**
- Describe the React framework
- Create React applications using `create-react-app`
- Construct reusable class and functional components
- Use JSX to bind data and create interactive components
- Manage state and props in an application
- Coordinate interaction between React components

**Outcome 3: Implement Routing and form validation in Angular and React**
- Implement routing with the Angular Router
- Validate forms using client-side validation in Angular
- Implement routing with React
- Validate forms using client-side validation in React

**Outcome 4: Deploy web applications to the MS Azure cloud**
- Describe cloud computing
- Create a Microsoft Azure account
- Create a pipeline to build and deploy code
- Execute CI/CD (Continuous Integration and Continuous Deployment)
- Describe the basic terms: subscription, account, and ARM
