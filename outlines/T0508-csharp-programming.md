# T0508 — C# Programming
**Program:** Full Stack Development (FSD)  
**Hours:** 40 hrs (2 weeks)  
**Delivery:** Online Instructor-Guided

---

## Description

Students gain a solid foundation in the C# programming language. Topics include the type system, control flow, methods, and object-oriented fundamentals. Students write, debug, and refactor C# console and class-library programs in preparation for ASP.NET Core and .NET courses that follow.

---

## Learning Outcomes

**Outcome 1: Write procedural C# programs using core language features**
- Identify C# syntax and program structure (namespaces, classes, `Main`)
- Declare variables using value types (`int`, `double`, `bool`, `char`) and `string`
- Use arithmetic, comparison, and logical operators
- Control program flow with `if/else`, `switch`, `for`, `while`, and `foreach`
- Define and call methods with parameters and return values
- Use `ref`, `out`, and optional parameters where appropriate
- Handle basic input and output with `Console.ReadLine` / `Console.WriteLine`

**Outcome 2: Apply object-oriented fundamentals in C#**
- Define classes with fields, properties, constructors, and methods
- Apply access modifiers (`public`, `private`, `internal`, `protected`)
- Implement auto-properties and expression-bodied members
- Use `static` members and utility classes
- Create and use enumerations and structs
- Instantiate objects; understand reference vs value semantics
- Apply basic exception handling with `try-catch`

## Course Schedule

### Module 1 — C# Language Fundamentals (Week 1, 20 hrs)

**Topics:**
- Introduction to .NET and C#: platform overview, SDK, IDE setup
- Program structure: namespace, class, `Main` entry point
- Variables and data types: primitives, `string`, `var` inference
- Operators: arithmetic, comparison, logical, compound assignment
- Control flow: `if/else`, `switch` expressions
- Loops: `for`, `while`, `do-while`, `foreach`
- Methods: signature, parameters, return types, overloading
- Arrays and `List<T>` basics
- String methods and interpolation
- Debugging in Visual Studio / VS Code

**Activities:**
- Write a program that reads user input and performs branching logic
- Implement a loop that processes an array of values
- Extract repeated logic into named methods
- Debug a program with intentional errors using breakpoints

**Assignment 1 due:** End of Week 1

---

### Module 2 — Object-Oriented Fundamentals (Week 2, 20 hrs)

**Topics:**
- Classes and objects: fields, constructors, methods
- Properties: auto-properties, full properties, computed properties
- Access modifiers and encapsulation
- Expression-bodied members and method overloading
- `static` members, utility classes, and the `Math` library
- Enumerations and `struct` types
- Value types vs reference types; `null` and nullability
- Basic exception handling: `try-catch-finally`, common exceptions
- Organising code: multiple classes in one project

**Activities:**
- Model a real-world entity as a C# class
- Add validation logic in a property setter
- Build a small console app with three or more interacting classes
- Handle a potential runtime exception gracefully

**Assignment 2 due:** End of Week 2
